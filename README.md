## Краткое описание работы

В ходе лабораторной работы были изучены основные управляющие структуры и типы данных в JavaScript:

### Что изучено:

**Циклы:**
- for (счётные циклы)
- while (циклы с предусловием)
- do...while (циклы с постусловием)
- вложенные циклы
- операторы break и continue

**Функции:**
- классическое объявление функций
- параметры и возвращаемые значения
- параметры по умолчанию
- функции как значения
- стрелочные функции (arrow functions)

**Массивы:**
- создание и доступ к элементам
- методы push(), pop()
- поиск элементов (includes(), indexOf())
- деструктуризация
- spread/rest операторы
- цикл for...of

**Объекты:**
- создание объектов (object literal)
- свойства и методы
- добавление и удаление свойств
- перебор свойств (for...in)
- вложенные объекты и массивы


## Выполненные практические задания

### Шаг 5. Циклы
- **№1:** Вывод чисел от 1 до 10, чётные числа, сумма чисел
- **№2:** Уменьшение переменной от 5 до 0 с помощью while
- **№4:** Вывод чисел от 1 до 10 с пропуском 5 и остановкой на 8
- **№5:** Вывод пирамиды из звёздочек с помощью вложенных циклов

### Шаг 6. Функции
- **№1:** Функция multiply для умножения двух чисел
- **№2:** Функция printInfo для вывода имени и возраста
- **№3:** Функция calculateDiscount с параметром по умолчанию

### Шаг 7. Массивы
- **№1:** Работа с массивом colors (доступ и изменение элементов)
- **№2:** Добавление и удаление элементов в массиве students
- **№4:** Поиск города в массиве (includes + indexOf)

### Шаг 8. Объекты
- **№1:** Создание объекта book и изменение свойств
- **№2:** Создание объекта car с методом getInfo()
- **№3:** Перебор свойств объекта product с помощью for...in

## Итоговые выводы

1. JavaScript имеет схожий с C# синтаксис для циклов и условий, что облегчает изучение
2. Функции в JavaScript более гибкие — могут быть присвоены переменным и переданы как аргументы
3. Массивы в JS могут содержать элементы разных типов и динамически изменять размер
4. Объекты можно создавать без классов и модифицировать на лету
5. Стрелочные функции предоставляют краткий синтаксис и удобны для коллбэков
